Three Single-Line Drawing Styles

Which direction feels right for the output you want?

A — Clean Contour

Minimal lines. Only essential outlines. High readability, iconic feel. Like those viral one-line portraits.

B — Gestural / Sketchy

Loose, expressive, imperfect. Lines wobble and cross. Feels hand-drawn and alive.

C — TSP / Tonal

One dense winding path builds up shading. Dark areas = dense loops, light areas = sparse. Photographic tonal feel.
