Design: New Pipeline Architecture

Section 1 of 4 — overall structure. The old flow is replaced from step 4 onward.

Old pipeline (current)

Grayscale → Levels → Posterize → Median → RegionBounds → ContourTrace → RDP → GreedyConnect → Chaikin → Bezier SVG

Red = root problem: region boundaries produce blocky grid edges; downstream steps can't recover artistic quality lost here.

New pipeline

Grayscale → Levels → Blur (σ-scaled) → Canny + Hysteresis → Zhang-Suen Skeleton → Junction Graph → DFS Path → Smooth Spline → SVG

Green = new or rewritten. The first two steps are unchanged.

What each new step does

Step Purpose Key change vs now
Blur (σ-scaled) Gaussian blur with σ = 0.006 × diagonal. Removes high-frequency texture while preserving structural edges. Was: fixed radius in pixels → broke on different image sizes
Canny + Hysteresis NMS already exists. Add double-threshold hysteresis: strong edges lock in, weak edges survive only if connected to a strong one. Produces clean, continuous contours. Was: posterize → regionBoundaries → blocky grid artifacts
Zhang-Suen Skeleton Iterative thinning until edges are 1-pixel-wide centerlines. Preserves topology (no disconnections). Required for principled graph construction. Was: contours already 1px (regionBoundaries), but structurally messy
Junction Graph Scan skeleton: classify each pixel as junction (3+ neighbors), endpoint (1 neighbor), or path pixel. Trace branches between junctions/endpoints. Prune branches shorter than 0.015 × diagonal. Was: no graph — greedy endpoint matching with no structure awareness
DFS Path Build MST of the graph weighted by branch length (longer = more important). DFS traversal of MST yields a single Euler tour where each branch is visited going in and coming back. At junctions: choose longest unvisited branch first. Add minimum-weight edges to make all nodes even-degree → guaranteed single closed loop. Was: greedy nearest-endpoint → chaotic jumps with no artistic coherence
Smooth Spline Fit Catmull-Rom spline through path points (replaces Chaikin corner-cutting). Tension parameter controls looseness. Output as cubic Bezier segments in SVG. Was: Chaikin doubles point count each pass → over-smoothed, loses character
