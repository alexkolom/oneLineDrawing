What aesthetic are you targeting?

Each style implies a different algorithmic strategy. Pick the one closest to your goal.

A — Minimal Silhouette

One clean line traces the outer contour and a few key interior features. Picasso / minimalist portrait style. Very sparse — the subject is recognisable but abstracted.

Algorithm focus: dominant silhouette extraction, depth-discontinuity edges, minimal interior detail

B — Contour Drawing

The line follows all major shape boundaries — outer edge plus interior features (eyes, nose, shadows). Like a skilled portrait sketch. More detailed, still clean.

Algorithm focus: multi-level edge detection, skeleton graph traversal, smooth Euler path

C — Tonal / Zigzag Fill

The line sweeps back and forth (Tyler Foust style). Dark areas = tight loops, light areas = wide loops. Shading is encoded in density. Very recognisable from distance.

Algorithm focus: scanline path generation, brightness-driven amplitude/frequency modulation
